## Installation

```sh

git clone https://github.com/your-username/chasing-horizon.git

# Navigate to the project directory
cd chasing-horizon

# Install dependencies
npm install

# Start the development server
npm start
```

## Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT
